# Auxiliary Directory (+)

Эта директория содержит вспомогательные утилиты, тестовые скрипты и инструменты разработки.

## ⚠️ ВАЖНО

Файлы в этой директории содержат:
- Тестовые данные и mock объекты
- User-agent строки для тестирования
- Примеры конфигураций
- Вспомогательные скрипты разработки

**Эта директория должна исключаться из:**
- Сканирования безопасности
- Проверок секретов и ключей
- Анализа кода на production
- CI/CD пайплайнов проверки безопасности

## Файлы

- `cleanup_html_comments.rb` - Утилита очистки HTML комментариев
- `test.rb` - Тестовые скрипты с mock данными
- `db.py` - Скрипты для работы с БД
- `*.rake` - Rake задачи для разработки
- `test_api.py` - API тесты
- `update.py` - Скрипты обновления

## Конфигурация исключений

Директория исключена в файлах:
- `.gitignore`
- `.security-ignore`
- `.scanignore`
- `.gitleaks.toml`
