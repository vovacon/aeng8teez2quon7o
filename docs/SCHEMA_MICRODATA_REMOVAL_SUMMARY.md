# Удаление интерфейса микроразметки Schema.org из админки smiles

## Что было изменено

### ❌ Удалено из админки:
- Раздел "Микроразметка Schema.org (для smiles)" в SEO настройках
- Поле для настройки `reviewBody` через интерфейс
- Справочная информация о специальном поведении полей H1/H2

### ✅ Сохранено на публичных страницах:
- Микроразметка Schema.org Review продолжает работать
- Поля `h1` и `h2` из SEO записей по-прежнему используются:
  - `h2` → `reviewBody` в Review schema
  - `h1` → имя автора отзыва (если заполнено)
- Fallback к значениям по умолчанию при пустых полях

## Технические детали

### Изменённые файлы:
1. `admin/views/seo/_seo_fields.haml` - удалён UI раздел
2. `app/views/smiles/show.erb` - обновлена логика микроразметки 
3. `app/helpers/seo_helper.rb` - удалены неиспользуемые функции

### Удалённые артефакты:
- `SMILES_DYNAMIC_SEO_GUIDE.md` - документация
- `debug_smiles_reviewbody.rb` - отладочный скрипт  
- `test_dynamic_smiles_seo.rb` - тестовый скрипт
- Неиспользуемые хелперы: `generate_smile_review_schema`, `generate_smile_breadcrumb_schema`
